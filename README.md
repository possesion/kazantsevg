# Kazantsev Gennadiy web 

## Описание
Этот сайт-портфолио предназначен для демонстрации моих способностей использования технологий в веб разработке и применения некоторый готовых технологий.

## Запуск приложения
### 1. Clone the project

### 2. Run the project
```shell
npm i
npm start
```

### 3. Build
```shell
npm run build
```
